# O Estúdio 🎤🎶

## **Overview**
O Estúdio is a live concert streaming platform designed for music lovers. Inspired by Tiny Concerts, it enables users to **discover**, **watch**, and **interact** with exclusive performances.

### **Core Features**
- 📽 **Stream Live & Recorded Concerts**
- 🔍 **Search & Filter by Artist, Genre, or Date**
- ❤️ **Like, Comment, and Favorite Videos**
- 🎤 **Artist Dashboard for Content Management**
- 🚀 **Admin Panel for Moderation & Analytics**

---

## **Technology Stack**
- **Frontend:** React (Next.js) – Dynamic & responsive UI
- **Backend:** Flask – API handling authentication & interactions
- **Database:** SQLite – Lightweight local data storage
- **Authentication:** JWT – Secure user management
- **CI/CD:** GitHub Actions – Automated testing & deployment
- **Hosting:** Vercel (Frontend) + Azure (Backend)
